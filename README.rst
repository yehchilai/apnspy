Quick Start
-----------

Use ``pip`` to install
.. code-block::bash
    $ pip install apnspy

(Optional) creat an virtual environment and use ``pip`` to install

.. code-block::bash
    $ virtualenv venv
    $ source venv/bin/activate
    $ pip install apnspy
