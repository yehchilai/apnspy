A Python client for Apple Push Notification service (under developing)

For more information, please read the `document`_

Quick Start
-----------

Use ``pip`` to install

.. code-block:: bash

    $ pip install apnspy


(Optional) creat an virtual environment and use ``pip`` to install

.. code-block:: bash

    $ virtualenv venv
    $ source venv/bin/activate
    $ pip install apnspy
